## Usage

1. Run `npm i`
2. Create an `.env.local` file in the root of the project.
3. In this new file, define `VITE_OAUTH_ISSUER` and `VITE_OAUTH_CLIENTID` with the appropriate credentials. 
4. Run `npm start`
